### 1.声明式程序设计和命令式程序设计有什么区别
命令式关注“如何做”某个任务，声明式关注“做什么”任务
### 2.函数式程序设计的基本特征
把程序组织成一组数学函数，计算过程体现为基于一系列函数应用的表达式求值。函数也被作为值来看待，即函数的参数和返回值也可以是函数。
引用透明、无副作用、无赋值操作、高阶函数、循环、延迟
### 3.尾递归调用相比一般递归函数实现有什么优点
可以有效的避免栈溢出，其运行时等效于循环
## 编程题
### 1.一只青蛙一次可以跳上1级台阶，也可以跳上2级。求该青蛙跳上一个n级的台阶总共有多少种跳法。可以定义f(n)表示青蛙跳上n级台阶的方法数。 请你用递归、尾递归、非递归的方式实现f，并输出n为3-10时的方法数。（可根据情况修改f的形参列表）
```cpp
//递归
int f1(int n){
    if(n <= 1){
        return 1;
    }
    return f1(n - 1) + f1(n - 2);
}

//尾递归
int f2(int n, int a, int b){
    if(n <= 1){
        return b;
    }
    return f2(n - 1, b, a + b);
}

//非递归
int f3(int n){
    if(n <= 1){
        return 1;
    }
    int a = 1, b = 2;
    for(int i = 3; i <= n; i++){
        b = a + b;
        a = b - a;
    }
    return b;
}
```

        extract(vector<S>(data.begin() + 1, data.end())).begin(),
        extract(vector<S>(data.begin() + 1, data.end())).end()
    ), extract(vector<S>(data.begin(), data.begin() + 1));
}
